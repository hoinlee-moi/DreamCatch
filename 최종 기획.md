# DreamCatch 개발 로드맵

## 📋 프로젝트 개요

**프로젝트명**: 드림캐치 (DreamCatch)

**영문 표기**: DreamCatch: AI Dream Archive & Gallery

**슬로건**: "당신이 잠든 사이 사라지는 조각들을 붙잡다"

**프로젝트 코드**: `Project-DC` (Dream Catcher)

### 핵심 가치

웨어러블 기반의 즉각적인 꿈 채집과 AI를 통한 시각적 재구성으로 개인의 무의식을 기록하는 통합 플랫폼

### 핵심 기술 필러 (Key Technical Pillars)

- **Catch (수집)**: Wear OS 및 Android 네이티브 기술을 통한 '기상 직후 초단기 기억'의 텍스트화
- **Convert (변환)**: AI를 활용한 '추상적 텍스트의 구체적 이미지화'
- **Curate (전시)**: Next.js 웹 플랫폼을 통한 '무의식 갤러리' 구축 및 소셜 공유 기능
- **Protect (보호)**: Redis와 JWT를 결합한 '보안 가용성 최적화'로 민감한 사생활 데이터 철저 보호

## 🏗️ 시스템 아키텍처

### 사용자 영역 (Client Side)

- 웹 브라우저 (Next.js 16.1.6)
- 안드로이드 앱 (Kotlin + Jetpack Compose)
- Wear OS 워치 (Kotlin + Wear Compose)

### 서버 영역 (Server Side)

- 백엔드 API 서버 (Next.js API Routes → Nest.js)

### 데이터 흐름

1. 사용자가 웹/모바일/워치에서 꿈 기록

3. AI 서버에서 텍스트 분석 및 이미지 생성 프롬프트 생성
